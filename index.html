Installing gulp

  1. Install the gulp cli globally

    $npm install --global gulp-cli

  2. Initialize your project directory:

    $ npm init

  3. Install gulp in your project devDependencies:

    npm install --save-dev gulp

  4. Create a gulpfile.js at the root of your project:

    var gulp = require('gulp');

    gulp.task('default', function() {
      // place code for your default task here
    });

  5. Run gulp

    gulp

    To run individual tasks, usegulp <task> <othertask>

Plugin: Gulp changed

Plugin That checks if a file has changed before it process it, saves alot of time. Get it here.

Install the plugin by using
sudo npm install --save-dev gulp-changed

Example:
gulp.task('changed', function(){
    return gulp.src(SRC)
   .pipe(changed(DEST))

   .pipe(yourTask)
   .pipe(gulp.dest(DEST)
); });

Note: .pipe(changed(DEST)) is needed to be able to compare the current files with the destination files.

Changed is really helpful in combination with the Watchtask.

Using the Watch Task

Watch files and do something when a file changes. This always returns an EventEmitter that emits change events.
